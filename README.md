# emlogit

<!-- badges: start -->

<!-- badges: end -->

`emlogit` implements the EM algorithm for the multinomial logistic
regression.

## Installation

You can install the development version from
[GitHub](https://github.com/) with:

``` r
# install.packages("devtools")
devtools::install_github("sou412/emlogit")
```
